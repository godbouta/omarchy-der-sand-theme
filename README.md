# Der-Sand Omarchy Theme

A liquid glass, minimalistic, desert-like theme for Omarchy desktop.

<img src="./preview.png" alt="Preview" width="100%">

## Install

Use the Omarchy theme installer:

```bash
omarchy-theme-install https://github.com/godbouta/omarchy-der-sand-theme
```
